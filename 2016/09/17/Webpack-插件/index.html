Webpack 插件

什么是 Webpack 插件

插件是对 javascript 代码进行逻辑处理的模块。

Config 配置型插件

  • NormalModuleReplacementPlugin 插件-模块替换

  • ContextReplacementPlugin 插件-上下文替换

关于 context，详见文档

  • IgnorePlugin

  • PrefetchPlugin

预加载模块以提升性能。

  • ResolverPlugin

  • ResolverPlugin.FileAppendPlugin

  • EnvironmentPlugin

设置可以通过 process.env 获取的环境变量。

Output 输出型插件

  • BannerPlugin

在打包后的块文件顶部添加额外内容。

Optimize 优化型插件

  • DedupePlugin

将文件去重以减小打包文件。去重后的模块只存在一个实例。

不建议在开发模式中运用。

  • LimitChunkCountPlugin

设置生成的块文件的最大数目。超过的文件会合并起来。

  • MinChunkSizePlugin

将阈值以下的块文件合并。

  • OccurrenceOrderPlugin
